Notes Table Schema:

CREATE TABLE notes (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    type ENUM('success','error','warning','info'),
    message LONGTEXT,
    is_read BOOLEAN DEFAULT FALSE,
    is_permanent BOOLEAN DEFAULT FALSE,
    auto_dismiss_in INT DEFAULT 5000,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    deleted_at TIMESTAMP NULL,
    INDEX idx_user_id (user_id)
);

Column Details:
  - id: Unique note identifier
  - user_id: Reference to users table
  - type: Color-coded note type
  - message: Note content
  - is_read: Read status tracking
  - is_permanent: Auto-dismiss vs persistent
  - auto_dismiss_in: Milliseconds before auto-dismiss
  - Timestamps: Created/updated/deleted tracking
